# Sales Analytics Dashboard

A Python Flask dashboard for visualizing product sales performance.

## Features
- Product sales visualization
- Bar chart with Matplotlib
- Flask web interface
- Sales performance overview

## Technologies
- Python
- Flask
- Matplotlib
- HTML
